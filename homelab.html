Project Overview

A comprehensive home lab environment designed to simulate enterprise network infrastructure for cybersecurity testing, learning, and skill development. This project serves as a practical platform for implementing security controls, testing configurations, and gaining hands-on experience with enterprise-grade technologies.

Architecture

Network Infrastructure

  • Segmented network with multiple VLANs
  • pfSense firewall for network security
  • Cisco networking equipment for enterprise simulation
  • DMZ for external-facing services

Security Components

  • IDS/IPS system using Security Onion
  • Network monitoring with Nagios
  • SIEM implementation with Splunk
  • Vulnerability scanning with Nessus

Infrastructure Services

  • Active Directory Domain Services
  • DNS and DHCP services
  • Certificate Authority
  • File and print services

Implementation Details

Virtualization Platform

  • VMware ESXi hypervisor
  • vSphere management
  • Resource allocation and monitoring

Network Security

  • VLAN segmentation
  • Access Control Lists
  • Network isolation

Monitoring & Logging

  • Centralized logging
  • Performance monitoring
  • Security alerts

Technologies Used

Networking

Infrastructure
