Home Lab Environment

Building an Enterprise Network
Active Development Last Updated: March 2024

Project Overview

A comprehensive home lab setup designed to simulate an enterprise network environment for cybersecurity testing, learning, and research. This environment provides hands-on experience with network security, system administration, and penetration testing.

Technologies Used

Networking
Security
Virtualization
Infrastructure
Monitoring

Lab Architecture

Network Infrastructure

  • Implemented VLANs for network segmentation
  • Configured routing and switching
  • Set up DMZ for external-facing services
  • Implemented NAT and firewall rules

Security Components

  • IDS/IPS deployment using Snort/Suricata
  • Security Information and Event Management (SIEM)
  • Vulnerability scanning with Nessus
  • Web Application Firewall (WAF)

System Infrastructure

  • Active Directory Domain Services
  • DNS and DHCP services
  • Certificate Authority
  • File and Print Services

Implementation Details

Phase 1: Network Setup

Established the foundational network infrastructure with proper segmentation and security controls.

  • Configured VLANs and trunking
  • Implemented routing protocols
  • Set up firewall rules and ACLs

Phase 2: Security Implementation

Deployed security tools and monitoring solutions to protect and observe the network.

  • Installed and configured IDS/IPS
  • Set up log aggregation and analysis
  • Implemented vulnerability scanning

Phase 3: Service Deployment

Deployed core services and ensured their security and availability.

  • Configured Active Directory
  • Implemented PKI infrastructure
  • Set up monitoring and alerting
